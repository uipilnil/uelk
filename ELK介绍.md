# ELK 介绍

## 什么是 ELK

ELK 是一个日志监控平台，由三个组件组成：

- **Elasticsearch**
    - **介绍**：这是一个分布式搜索引擎，用于存储数据，并对其进行搜索和分析；
    - **在 ELK 体系中**：它负责为日志 **建立索引**，并进行 **存储**，方便业务层对日志进行检索。
- **Logstash**
    - **介绍**：这是一个用来处理数据的管道；
    - **在 ELK 体系中**：它负责对日志进行 **收集**、**分析**、**过滤**，然后把日志 **转发给 ElasticSearch**；
    - 特点：
        - 它可以 **动态** 采集 **多个数据源** 的数据。
- **Kibana**
    - **介绍**：这是一个 **数据分析和可视化** 平台，它把数据以图表、地图、仪表盘等格式展现出来；
    - **在 ELK 体系中**：虽然它是为 Elasticsearch 设计的，但它也为 Logstash 提供 Web 界面。



三者的配合关系：

Logstash 把日志 -> Elasticsearch。Kibana 提供可视化界面。
